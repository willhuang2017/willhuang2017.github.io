HTML Reference

Common HTML Tags

Tag Name Description
<!DOCTYPE> Doc Type Allows Browser to know what type of document to expect.
<html> HTML Container for all HTML code.
<head> Head Container for all HTML metadata.
<title> Title Displays title on tab of webpage.
<body> Body Container for all visible HTML code.
<h[1-6]> Header Headings h1 is the largest, h6 is the smallest.
<div> Division Container that divides the page into sections.
<p> Paragraph Contains plain text.
<span> Span Seperates pieces of content inline.
<em> Emphasis Dispalys text in italics.
<strong> Strong Dispalys text in bold.
<br> Break Line break.
<ul> Unordered List Container for unordered list of items.
<ol> Ordered List Container for ordered list of items.
<li> List An item in a list.
<img /> Image Displays an image.
<video> Video Displays a video.
<a> Anchor Links to another webpage.

Global Attributes

Attribute Name Description
id Id Specifies a unique id.
class Class Specifies class for one or more elements.

Image & Video Attributes

Attribute Name Description
src Source Specifies the location of the image or video.
alt Alternative Text Provides alternative text if image is inaccessible.
width Width Provides width of the video.
length Length Provides length of the video.
controls Controls Provides basic playback features.

Anchor Attributes

Attribute Name Description
href Hyperlink Reference Links to the resource, use #id for in-page links
target Target Determines how the link should be opened.
title Title Hover text of link.

CSS Reference

Position Values

Value Description
static Element in default position
relative Element positioned relative to its default static position
absolute Element positioned relative to its nearest parent element
fixed Element positioned to specific point on the page
sticky Element is in user flow but then sticks after enough scroll
